# Hammerspoon-X

Mac 上的快捷指令

### **Hammerspoon安装**

 `brew install --cask hammerspoon`



### **配置**

![image-20210604205614991](/README.assets/image-20210604205614991.png)



把项目中的 `init.lua` 复制到 Config 里面即可



### **快捷键**

默认组合键 mash =「Control + Option + Command」

当前窗口全屏     `mash + m`

当前窗口最大化 `mash + l`

当前窗口半屏幕显示 `mash + 方向键上下左右`

当前窗口移动到下一个屏幕  `mash + n`
